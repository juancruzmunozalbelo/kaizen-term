KaizenTerm

The first Terminal Orchestration Cockpit for the Vibe Coding era.

Run Claude Code, Codex CLI, Aider, and OpenCode — all at once.

⭐ View on GitHub ↧ Download ZIP

Why KaizenTerm?

🤖

Multi-Agent Orchestration

Run Claude Code, Codex CLI, Aider, OpenCode, or any terminal-based AI agent simultaneously in separate panes.

📄

Integrated Kanban Board

Create tasks, assign them to agents, and watch them flow from Backlog to Done automatically.

💬

Natural Language Commands

Type # followed by plain English. KaizenTerm translates it to the right shell command.

📡

@Mention Agents

Reference multiple agents in a single prompt: @frontend @backend why CORS?

⏱

Focus Timer

Built-in Pomodoro timer to keep you in deep work, right where you code.

🔌

MCP Integration

Model Context Protocol server provides rich project context to your AI tools.

Get Started

# Clone the repo
git clone https://github.com/juancruzmunozalbelo/kaizen-term.git
cd kaizen-term

# Install and run
npm install
npm run dev
